### Rh2SU
### 
犀牛导出su脚本，简单易用，导出su模型的模块各自成组，各自拥有相对独立的原点，并自动整理乱线。解决了犀牛导出su的破面、原点过远问题。

**具体使用方法：**  
**1.在犀牛指令栏输入：RunPythonScript  
2.选择脚本：Rh2SU_LZ_beta02.py运行  
3.按提示选择物件操作  
4.导出为.skp文件**  


***PS:如果要导出分别存在于多个图层的复杂模型文件，建议:  
1.新建Rhino文件  
2.将模型放入同一个默认图层再运行脚本(此操作是为防止由于图层嵌套造成bug闪退)***
